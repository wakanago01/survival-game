# **survival game（仮）**

## ジャンル
2Dドット絵サバイバルクラフト × スローライフゲーム
# ストーリー
主人公は、人間によく似た姿を持つ**地球外生命体**。
平和に暮らしていた惑星へ、資源を狙う地球人が侵略してきた。
主人公は故郷を守りながら、いつか仲間と共に**ロケットを完成させ、土星へ移住する**ことを目標に行動する。
# ゲームの最終目標
ロケットを完成させ、**土星へ到達すること。**
ロケットは一度に完成するものではなく、多くのパーツを集めながら少しずつ組み立てていく。
# 基本サイクル
① ファームをする
↓
② お金・素材・部品を集める
↓
③ 武器・装備をクラフトする
↓
④ 地球人基地へ挑戦する
↓
⑤ ロケットパーツを入手する
↓
⑥ ロケットを強化する
↓
⑦ 土星へ出発（ゲームクリア）
# フィールド
## 主人公の惑星
平和で幻想的な惑星。
特徴
* 宇宙植物が育つ畑
* 光る森林
* 鉱山
* 湖・川・海
* 釣りスポット
* 地球人基地
空には巨大な惑星や流星が見え、昼夜で景色が変化する。
# ファーム要素
プレイヤーは自由に生活を楽しめる。
## 農業
* 宇宙植物を育てる
* 水やり
* 収穫
* 種を購入
収穫時に低確率で
* 小さな電子部品
* ネジ
* 回路
* エネルギー結晶
などロケット素材が手に入る。
## 採掘
鉱山で
* 鉱石
* 宝石
* レア鉱石
を採取できる。
クラフト素材にもなる。
## 釣り
宇宙魚を釣る。
魚は
* 売却
* 食料
* クラフト素材
として利用できる。
まれに小型部品が釣れる。
# 戦闘
敵は**地球人**。
普段は戦闘にならない。しかし、
**地球人基地の周囲へ近づくと戦闘エリアに入り、敵が出現する。**

# 報酬
基地攻略で
* ロケットの大型パーツ
* エンジン部品
* 燃料タンク
* コンピューター
* 特殊素材
が入手できる。
ロケット完成には基地攻略が必須。

# 武器
クラフトで作成。
例
* 光の剣
* レーザーライフル
# クラフト
作れるもの
* 武器
* 防具
* 農具
* 釣り竿
* ロケット部品
* 家具
素材は
* 農業
* 採掘
* 釣り
* 戦闘
すべてから集められる。

# ショップ
ファームで得たものを販売できる。
販売

* 作物
* 魚
* 鉱石
* 宝石

購入

* 食べ物
* 武器
* 種
* 鉱石
* クラフト素材
* 釣り道具
* 回復アイテム

# 一日の流れ

1日は**40分**。

* 昼：20分
* 夜：20分

夜は

* 一部の宇宙生物が出現
* レア素材が採取可能

# 睡眠

主人公は家で眠れる。寝ると
* HP全回復
* 次の日になる
睡眠はゲーム進行の区切りとなる。

# HP・ゲームオーバー
HPが0になるとゲームオーバー。
ペナルティとして**3日前の朝から再開。**
失われるものがあるため、探索や基地攻略では慎重な行動が求められる。

# UI
表示内容
* HP
* 所持金
* 時刻
* 日数
* ショートカットアイテム
* ロケット完成率
* ミニマップ（任意）
